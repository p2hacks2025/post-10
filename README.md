## 対象ユーザ
- 普段からSNSを利用していて、論争や炎上を見るのに辟易した人
- 良い思い出にたくさん触れたい人

## 利用の流れ
**ブラウザでURLを開くだけ！**  
隙に投稿してもよし、他人の投稿を見るもよし！  
「いいね」が沢山ついた投稿は光って目立つ！  
「よくないね」が沢山つくと見えなくなってくる！

## 推しポイント
タイムラインに流れてくるのは、キラキラした思い出や面白い投稿などばかり！  
自分たちで綺麗なSNSを作れる！  
 ~~妬ましい投稿に「よくないね」をつけて見えなくすることも~~
