## 利用の流れ
**ブラウザでURLを開くだけ！**  
隙に投稿してもよし、他人の投稿を見るもよし！  
「いいね」が沢山ついた投稿は光って目立つ！  
「よくないね」が沢山つくと見えなくなってくる！

## 推しポイント
